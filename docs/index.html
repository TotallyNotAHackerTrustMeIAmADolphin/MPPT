Voltage Calibration

Connect your power supply to both the input and output terminals.

Adjust the supply to exactly 15.0 V (or close to it).

Note: Ensure the bridge is solid before proceeding.

Step 1: Voltage Low Point

Set supply to ~15.0 V and enter the exact reading:

Vin Raw: 0 | Vout Raw: 0

Step 2: Voltage High Point

Set supply to ~24.0 V and enter the exact reading:

Vin Raw: 0 | Vout Raw: 0

Voltage Calibration Complete!

Click below to save the voltage calibration to EEPROM.

Current Calibration

Connect supply to Input and an electronic load to Output.

WARNING: Clicking 'Next' will activate Pass-through mode (MOSFETs ON).

Step 1: Current Low Point

Set load to ~0.50 A and enter the exact reading:

Ain Raw: 0 | Aout Raw: 0

Step 2: Current High Point

Set load to ~2.00 A and enter the exact reading:

Ain Raw: 0 | Aout Raw: 0

Current Calibration Complete!

Click below to save the current calibration to EEPROM.
